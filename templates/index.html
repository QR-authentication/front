QR Authentication

About QR Authentication

This project provides a secure and user-friendly QR-based authentication system. It allows users to generate and scan QR codes for seamless authentication, ensuring security and convenience.

Key features include:

  • QR code generation for secure token distribution
  • Real-time QR scanning using device cameras
  • Robust authentication with JWT-based cookies
  • Responsive design for desktops and mobile devices

Developed with modern web technologies, this project aims to demonstrate secure authentication methods using QR codes, suitable for various applications like login systems, access control, and more.
